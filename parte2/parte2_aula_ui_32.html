Section 1

A tag P é uma tag HTML que indica um parágrafo. Ela é usada para indicar o início e o fim de um bloco de texto que forma um parágrafo. Quando usada corretamente, essa tag permite que os navegadores da web exibam o conteúdo de uma página da web de forma estruturada e organizada, facilitando a leitura e a compreensão do conteúdo pelo usuário.

Section 2

A tag Span é uma tag HTML usada para aplicar estilos ou outras propriedades a uma parte específica do texto dentro de um elemento maior. Ao contrário da tag P, que é usada para delimitar um parágrafo inteiro, a tag Span é usada para delimitar uma parte menor de um texto, como uma palavra ou uma frase.
